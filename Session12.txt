class and id
-----------------------
id:
    1) one id can be given to only one element.
    2) one element can have only one id.

class:
    1) one class can be given to multiple elements.
    2) one element can have multiple classes.

<p class="para red desc">this is a paragraph</p>

<p class="para">this is another paragraph</p>

<h3 class="para">this is heading</h3>
